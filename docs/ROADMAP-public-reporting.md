# ระบบรายงานจากประชาชน

> สถานะ: **กำลังพัฒนา** | v5.2.0
> Admin สามารถเปิด/ปิดระบบนี้ได้

## แนวคิดหลัก
- ประชาชนรายงานสถานะน้ำมัน "มี" หรือ "หมด"
- ถ้าหลายคนรายงานเหมือนกัน → น่าเชื่อถือมากขึ้น (เช่น "3 คนรายงานว่าดีเซลหมด")
- Staff รายงาน → เชื่อถือ 100% ทันที
- Admin เปิด/ปิดฟีเจอร์นี้ได้จาก Dashboard

## 1. ระบบรายงาน (แบบง่าย)
- ประชาชน login ด้วย **LINE Login** อย่างเดียว (ไม่มีสมัครสมาชิกแยก)
- ได้ LINE user ID, ชื่อ, รูปโปรไฟล์ มาเก็บในระบบ
- กดปุ่ม "รายงาน" → แผนที่ zoom ไปตำแหน่ง GPS ของผู้ใช้ → แสดงปั๊มที่อยู่ใกล้ ≤500m
- เลือกปั๊มจาก map โดยตรง → เลือกชนิดน้ำมัน → รายงาน "มี/หมด"
- ถ้าไม่มีปั๊มในรัศมี 500m → แจ้ง "ไม่พบปั๊มใกล้คุณ"
- ระบบนับจำนวนคนที่รายงานเหมือนกัน
- แสดง "N คนรายงาน" ให้คนอื่นเห็น

## 2. ค่าความเชื่อมั่น
```
รายงานจาก Staff            → 100% (ไม่หมดอายุ)
รายงาน 1 คน (GPS ≤500m)   → 60%
รายงาน 2 คนเหมือนกัน      → 80%
รายงาน 3+ คนเหมือนกัน     → 90%
```
> ต้องอยู่ใกล้ปั๊ม ≤500 เมตร ถึงรายงานได้ (บังคับ GPS)

## 3. ป้องกัน Spam
| วิธี | รายละเอียด |
|------|-----------|
| GPS บังคับ | ต้องอยู่ใกล้ปั๊ม ≤500 เมตร ถึงรายงานได้ (ไกลกว่า = บล็อก) |
| Rate limit | รายงานได้ 5 ครั้ง/ชม. ต่อคน |
| LINE Login | ต้อง login ด้วย LINE ก่อนรายงาน |
| รูปถ่าย | ถ่ายรูปประกอบ (optional แต่ได้ confidence เพิ่ม) |

## 4. หมดอายุอัตโนมัติ
- รายงานจาก **Staff** → ไม่หมดอายุ
- รายงานจาก **ประชาชน** เก่ากว่า **2 ชม.** → ลดความเชื่อมั่น
- รายงานจาก **ประชาชน** เก่ากว่า **6 ชม.** → หมดอายุ (สถานะ "ไม่ทราบ")

## 5. Reputation (อนาคต)
- คนที่รายงานถูกต้องบ่อย → คะแนนสูง → รายงานเชื่อถือได้ทันที
- คนที่รายงานผิดบ่อย → คะแนนต่ำ → ต้องรอคนอื่นรายงานเหมือนกัน
